# Linea

Minimalist jewelry e-commerce storefront for curated collections, product detail pages, cart review, checkout, and brand content.

## Tech Stack

- Vite
- React 18 + TypeScript
- Tailwind CSS
- shadcn/ui (Radix UI primitives)
- React Router

## Local Development

```sh
npm install
npm run dev
```

## Build

```sh
npm run build
```

## Lint

```sh
npm run lint
```
